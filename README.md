# Time Tracking App

## Introduction

Time Tracking App is a productivity tool designed to help users monitor, manage, and analyze their time usage. It enables individuals to keep accurate records of hours worked, tasks completed, and projects handled.

## Features

- Manual entry and editing of time logs
- Project and task management with categorization
- Responsive design for desktop and mobile devices

## Requirements

- Flutter SDK installed and added to your system path
- Dart SDK bundled with the Flutter installation
- Android Studio or Visual Studio Code with Flutter extensions
- Android emulator, iOS simulator, or a physical device for testing
- Git installed for cloning the repository

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Knighthawk213/Time_Tracking_App.git
   cd Time_Tracking_App
   ```

2. **Fetch Flutter dependencies**

   ```bash
   flutter pub get
   ```

3. **Verify Flutter setup**

   ```bash
   flutter doctor
   ```

4. **Run the app on a connected device or emulator**

   ```bash
   flutter run
   ```
